Bootstrap Images
================
Bootstrap support three types of images.

1) rounded image 

2) circle image 

3) Thumbnail image 


1) rounded image 
-----------------
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	
	<!-- add bootstrap plugins -->
	<meta charset="utf-8">

    <meta name="viewport" content="width=device-width, initial-scale=1">

     <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

     <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" ></script>
</head>
<body>
		<div class="container-fluid mt-5">
			<img src="images/profile.jpg" class="w-25 h-25 rounded" />
		</div>
</body>
</html>

Bootstrap tables 
=================
A tale is a collection of rows and columns.

A table is used to represent our data in the form of rows and columns.

To create a table in a bootstrap we need to use ".table" class.

ex:
---
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>
	
	<!-- add bootstrap plugins -->
	<meta charset="utf-8">

    <meta name="viewport" content="width=device-width, initial-scale=1">

     <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

     <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" ></script>
</head>
<body>
		<div class="container mt-5">
			
			<table class="table">
				<thead>
					<tr>
						<th>Id</th>
						<th>Name</th>
						<th>Address</th>
					</tr>
				</thead>
				<tbody>
					<tr>
						<td>101</td>
						<td>Alan</td>
						<td>Florida</td>
					</tr>
					<tr>
						<td>102</td>
						<td>Jose</td>
						<td>Texas</td>
					</tr>
					<tr>
						<td>103</td>
						<td>Nelson</td>
						<td>Chicago</td>
					</tr>

				</tbody>
				
			</table>
		</div>
</body>
</html>

Bootstrap Alerts
=====================
Boostrap has given a simple way to create alerts by using "alert" class along with contextual state classes.

We have following contextual state classes

ex:
	alert-success
	alert-danger
	alert-warning
	alert-info 

ex:1
------
<!DOCTYPE html>
<html>
<head>
	<title>MyPage!</title>

	<!-- Add bootstrap -->
<meta charset="utf-8">

<meta name="viewport" content="width=device-width, initial-scale=1">

<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">
<!-- Option 1: Bootstrap Bundle with Popper -->

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-MrcW6ZMFYlzcLA8Nl+NtUVF0sA7MsXsP1UyJoMp4YLEuNSfAP+JcXn/tWtIaxVXM" crossorigin="anonymous"></script>
	
</head>
<body>
	<div class="container mt-5">
		
		<div class="alert alert-warning">
			<strong>Warning!</strong> Sit properly
		</div>

		<div class="alert alert-danger">
			<strong>Danger!</strong> Online people understood na?
		</div>

		<div class="alert alert-info">
			<strong>Job!</strong> Tomorrow is a reactjs drive.
		</div>

		<div class="alert alert-success">
			<strong>Good News!</strong> We have proper placements.
		</div>

		
	</div>
</body>
</html>

=====================

We have following contextual state classes.

ex:
	btn-default
	btn-primary
	btn-secondary
	btn-info
	btn-warning
	btn-danger
	btn-success
	btn-light
	btn-link
	and etc.
